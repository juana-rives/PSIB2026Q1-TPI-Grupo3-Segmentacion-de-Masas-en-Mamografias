# Datos

Las imágenes DICOM, ROI y máscaras de referencia de CBIS-DDSM no se incluyen en este repositorio por su tamaño.

Para ejecutar el notebook, se espera una estructura local o en Google Drive equivalente a:

```text
TP-PSIB/
└── TPI/
    ├── IMG/
    │   ├── MG/
    │   ├── ROI/
    │   └── MASK/
    └── mass_case_description_train_set.csv
```

La ruta principal debe configurarse en el notebook mediante la variable BASE_DIR.
